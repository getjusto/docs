Lista los códigos asociados a un cupón ecommerce de la marca autorizada. Sirve para auditar uso, revisar códigos disponibles y buscar códigos específicos por texto.

La respuesta viene paginada. Para descargas completas desde Justo Admin MCP, usa `generate_data_file`; para una vista rápida usa `get_data`.
