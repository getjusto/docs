Lista entregas sin pedido creadas desde Justo Admin para la marca autorizada. Sirve para monitorear despachos manuales, buscar por código o externalId, y revisar entregas activas.

La respuesta viene paginada. Para descargas completas desde Justo Admin MCP, usa `generate_data_file`; para una vista rápida usa `get_data`.
