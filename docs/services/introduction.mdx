---
title: Servicios
---

APIs generales de Justo para integraciones que no operan directamente sobre una marca o website.

Esta sección agrupa servicios como Reservas, Embedded Orders y Ordering API. A diferencia de PDV, Ecommerce, Config, Delivery y Royalty, estas APIs se usan para habilitar capacidades transversales o flujos embebidos de Justo.
