Actualiza una dirección del customer.

Todas las llamadas usan `Authorization: Bearer <token>` con scope `orderingApi`.

## Endpoint

```http
PATCH /v3/ordering/customers/{externalCustomerId}/addresses/{addressId}
```

## Path parameters

<ParamField name="externalCustomerId" type="string" required>
  ID del customer en el sistema externo. Es independiente por app.
</ParamField>

<ParamField name="addressId" type="string" required>
  ID de una dirección guardada por el customer.
</ParamField>

## Body

<ParamField name="placeId" type="string" required>
  Place ID normalizado usado para cobertura y despacho.
</ParamField>

<ParamField name="address" type="string" required>
  Dirección principal visible para el customer.
</ParamField>

<ParamField name="addressLine2" type="string">
  Departamento, oficina u otra información secundaria.
</ParamField>

<ParamField name="addressSecondary" type="string">
  Texto secundario de comuna, ciudad o referencia.
</ParamField>

<ParamField name="location" type="object" required>
  Coordenadas geográficas con `lat` y `lng`.
</ParamField>

<ParamField name="acceptsNoLine2" type="boolean">
  Indica si el customer confirmó que no tiene línea 2.
</ParamField>

<ParamField name="comment" type="string">
  Comentario o instrucciones de entrega.
</ParamField>

<ParamField name="streetAddress" type="string">
  Calle y numeración normalizada.
</ParamField>

<ParamField name="locality" type="string">
  Localidad o comuna.
</ParamField>

<ParamField name="subLocality" type="string">
  Sub-localidad.
</ParamField>

<ParamField name="region" type="string">
  Región o estado.
</ParamField>

<ParamField name="postalCode" type="string">
  Código postal.
</ParamField>

<ParamField name="countryCode" type="string">
  Código de país ISO 3166-1 alpha-2.
</ParamField>

<ParamField name="countryName" type="string">
  Nombre del país.
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indica si la llamada fue procesada correctamente.
</ResponseField>

<ResponseField name="data" type="OrderingAddress" required>
  Dirección actualizada.
</ResponseField>
