Lista productos favoritos del customer en el website.

Todas las llamadas usan `Authorization: Bearer <token>` con scope `orderingApi`.

## Endpoint

```http
GET /v3/ordering/customers/{externalCustomerId}/websites/{domain}/products/favorites
```

## Path parameters

<ParamField name="externalCustomerId" type="string" required>
  ID del customer en el sistema externo. Es independiente por app.
</ParamField>

<ParamField name="domain" type="string" required>
  Dominio canónico del website Justo, sin protocolo ni `www.`.
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indica si la llamada fue procesada correctamente.
</ResponseField>

<ResponseField name="data" type="OrderingCartItem[]" required>
  Productos favoritos del customer en el website.
</ResponseField>

## Data object

### OrderingCartItem

Item del carrito.

<ResponseField name="_id" type="string">
  ID del item del carrito.
</ResponseField>

<ResponseField name="productId" type="string">
  ID del producto.
</ResponseField>

<ResponseField name="menuId" type="string | null">
  ID del menú.
</ResponseField>

<ResponseField name="amount" type="number">
  Cantidad del producto.
</ResponseField>

<ResponseField name="quantity" type="number">
  Cantidad del producto en flujos v2.
</ResponseField>

<ResponseField name="comment" type="string | null">
  Comentario del item.
</ResponseField>

<ResponseField name="modifiers" type="object[]">
  Modificadores seleccionados.
</ResponseField>

<ResponseField name="product" type="object">
  Producto asociado al item.
</ResponseField>

<ResponseField name="unitPrice" type="number">
  Precio unitario.
</ResponseField>

<ResponseField name="totalPrice" type="number">
  Precio total del item.
</ResponseField>
