Actualiza cantidad, modificadores o comentario de un item del carrito.

Todas las llamadas usan `Authorization: Bearer <token>` con scope `orderingApi`.

## Endpoint

```http
PATCH /v3/ordering/customers/{externalCustomerId}/websites/{domain}/cart/items/{cartItemId}
```

## Path parameters

<ParamField name="externalCustomerId" type="string" required>
  ID del customer en el sistema externo. Es independiente por app.
</ParamField>

<ParamField name="domain" type="string" required>
  Dominio canónico del website Justo, sin protocolo ni `www.`.
</ParamField>

<ParamField name="cartItemId" type="string" required>
  ID del item del carrito.
</ParamField>

## Body

<ParamField name="productId" type="string" required>
  ID del producto.
</ParamField>

<ParamField name="menuId" type="string">
  ID del menú desde el que se agrega el producto.
</ParamField>

<ParamField name="amount" type="number" required>
  Cantidad del producto. Debe ser mayor a 0.
</ParamField>

<ParamField name="modifiers" type="array">
  Selecciones de modificadores con `modifierId` y `optionsIds`.
</ParamField>

<ParamField name="comment" type="string">
  Comentario del item.
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indica si la llamada fue procesada correctamente.
</ResponseField>

<ResponseField name="data" type="OrderingCartItemMutation" required>
  Item actualizado y estado actualizado del carrito.
</ResponseField>
