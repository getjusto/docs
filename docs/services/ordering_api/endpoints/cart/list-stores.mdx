Lista locales disponibles según website, dirección y tipo de entrega.

Todas las llamadas usan `Authorization: Bearer <token>` con scope `orderingApi`.

## Endpoint

```http
GET /v3/ordering/customers/{externalCustomerId}/websites/{domain}/cart/stores
```

## Path parameters

<ParamField name="externalCustomerId" type="string" required>
  ID del customer en el sistema externo. Es independiente por app.
</ParamField>

<ParamField name="domain" type="string" required>
  Dominio canónico del website Justo, sin protocolo ni `www.`.
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indica si la llamada fue procesada correctamente.
</ResponseField>

<ResponseField name="data" type="OrderingStore[]" required>
  Locales disponibles para el carrito.
</ResponseField>

## Data object

### OrderingStore

Local disponible para operar pedidos.

<ResponseField name="_id" type="string">
  ID del local.
</ResponseField>

<ResponseField name="websiteId" type="string">
  ID del website al que pertenece el local.
</ResponseField>

<ResponseField name="name" type="string">
  Nombre del local.
</ResponseField>

<ResponseField name="timezone" type="string">
  Zona horaria del local.
</ResponseField>

<ResponseField name="acceptDelivery" type="boolean">
  Indica si acepta despacho.
</ResponseField>

<ResponseField name="acceptGo" type="boolean">
  Indica si acepta retiro.
</ResponseField>

<ResponseField name="acceptServe" type="boolean">
  Indica si acepta consumo en local.
</ResponseField>

<ResponseField name="acceptReservations" type="boolean">
  Indica si acepta reservas.
</ResponseField>

<ResponseField name="acceptWaitingList" type="boolean">
  Indica si acepta lista de espera.
</ResponseField>

<ResponseField name="minimumAnticipationDays" type="number">
  Días mínimos de anticipación para pedidos programados.
</ResponseField>

<ResponseField name="hideTip" type="boolean">
  Indica si se oculta la propina.
</ResponseField>

<ResponseField name="forceHideDeliveryTip" type="boolean">
  Indica si se fuerza ocultar propina de delivery.
</ResponseField>

<ResponseField name="defaultsManagedDelivery" type="boolean">
  Indica si el local usa despacho gestionado por defecto.
</ResponseField>

<ResponseField name="deliveryCostType" type="string">
  Tipo de cálculo de costo de despacho.
</ResponseField>

<ResponseField name="cityId" type="string">
  ID de la ciudad marketplace.
</ResponseField>

<ResponseField name="address" type="OrderingStoreAddress">
  Ver `OrderingStoreAddress`.
</ResponseField>

<ResponseField name="humanSchedule" type="OrderingHumanSchedule[]">
  Horario de atención legible.
</ResponseField>

<ResponseField name="reservationsHumanSchedule" type="OrderingHumanSchedule[]">
  Horario de reservas legible.
</ResponseField>

<ResponseField name="zones" type="object[]">
  Zonas de despacho del local.
</ResponseField>

<ResponseField name="availablePaymentMethodInfo" type="OrderingPaymentMethod[]">
  Medios de pago disponibles para el local.
</ResponseField>

<ResponseField name="supportOptions" type="object">
  Canales de soporte del local.
</ResponseField>
