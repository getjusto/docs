Obtiene el detalle de un producto, incluyendo disponibilidad y modificadores.

Todas las llamadas usan `Authorization: Bearer <token>` con scope `orderingApi`.

## Endpoint

```http
GET /v3/ordering/websites/{domain}/products/{productId}
```

## Path parameters

<ParamField name="domain" type="string" required>
  Dominio canónico del website Justo, sin protocolo ni `www.`.
</ParamField>

<ParamField name="productId" type="string" required>
  ID del producto dentro del catálogo Justo.
</ParamField>

## Query parameters

<ParamField name="menuId" type="string">
  ID del menú a consultar. Si se omite, se usa el menú por defecto del website.
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indica si la llamada fue procesada correctamente.
</ResponseField>

<ResponseField name="data" type="OrderingProduct" required>
  Detalle del producto en el menú consultado.
</ResponseField>

## Data object

### OrderingProduct

Producto de menú.

<ResponseField name="_id" type="string">
  ID del producto.
</ResponseField>

<ResponseField name="name" type="string">
  Nombre del producto.
</ResponseField>

<ResponseField name="description" type="string | null">
  Descripción del producto.
</ResponseField>

<ResponseField name="externalId" type="string | null">
  ID externo configurado por el comercio.
</ResponseField>

<ResponseField name="priority" type="number">
  Prioridad de ordenamiento.
</ResponseField>

</ResponseField>

<ResponseField name="availabilityAt" type="object">
  Precio y disponibilidad del producto en el menú consultado.
</ResponseField>

  Categorías del producto.

  Imágenes del producto.
</ResponseField>

<ResponseField name="maxPurchaseQuantity" type="number | null">

</ResponseField>

<ResponseField name="websiteId" type="string">
  ID del website.
</ResponseField>

  Nombre del website.
</ResponseField>

<ResponseField name="menuId" type="string">
  ID del menú consultado.
</ResponseField>

<ResponseField name="urlVideo" type="string | null">
  URL de video del producto.
</ResponseField>

<ResponseField name="modifiers" type="OrderingProductModifier[]">
  Modificadores del producto.
</ResponseField>

<ResponseField name="altImg" type="string | null">
  Texto alt de imagen.

<ResponseField name="titleImg" type="string | null">
  Título de imagen.

<ResponseField name="measurementUnit" type="string | null">

<ResponseField name="measurementValue" type="number | null">

<ResponseField name="translations" type="object | null">
