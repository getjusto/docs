Obtiene la configuración dinámica de facturación del website.

Todas las llamadas usan `Authorization: Bearer <token>` con scope `orderingApi`.

## Endpoint

```http
GET /v3/ordering/websites/{domain}/billing-params
```

## Path parameters

<ParamField name="domain" type="string" required>
  Dominio canónico del website Justo, sin protocolo ni `www.`.
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indica si la llamada fue procesada correctamente.
</ResponseField>

<ResponseField name="data" type="OrderingBillingParams" required>
  Parámetros y campos dinámicos de facturación.
</ResponseField>

## Data object

### OrderingBillingParams

Parámetros de facturación requeridos por el website.

<ResponseField name="billingParamsSchema" type="object[]">
  Schema dinámico de campos de facturación.
</ResponseField>

<ResponseField name="orderBillingRequired" type="boolean">
  Indica si la facturación es obligatoria.
</ResponseField>

<ResponseField name="countryCode" type="string">
  Código de país del website.
</ResponseField>
