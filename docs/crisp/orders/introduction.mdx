---
title: Empezar
description: 'Puedes realizar una integración propia para un canal de ventas que no este soportado directamente por Crisp.'
---

<Steps>
  <Step title="Crear un nuevo canal de ventas Custom">
    Anda a Ajustes > Canales de ventas y haz clic en el botón "Custom".
  </Step>
  <Step title="Define un token de seguridad">
    Define un token de seguridad para tu canal de ventas. Este token será usado
    para autenticar las solicitudes de tu canal de ventas. El token se usará
    para autenticar las llamadas hacia Crisp y desde Crisp a los webhooks que
    definas.
  </Step>
  <Step title="Guarda las urls de los webhooks">
    Si es que vas a usar webhooks, guarda las urls de los webhooks que vas a
    usar para recibir eventos de Crisp.
  </Step>
</Steps>

La integración tiene dos categorías. Primero las llamadas que haces desde tu servidor a Crisp (Endpoints) y luego las llamadas que hace Crisp a tu servidor (Webhooks).

Para realizar la integración más básica solo es necesario que uses el endpoint de Crear Pedido.
