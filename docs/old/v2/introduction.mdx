---
title: Empezar
description: 'Puedes comenzar creando tu token de autorización'
---

<Steps>
  <Step title="Activa el módulo de API">
    Solicita por el chat de soporte o a tu ejecutivo asignado para que te habiliten el módulo de API.
  </Step>
  <Step title="Ingresa a la sección de API Keys">
    Entra a tu portal GetJusto y dirígete a tu menú lateral izquierdo y haz clic sobre `Plataforma > API` y selecciona `API Keys`.
  </Step>

  <Step title="Define un token de seguridad">
    Una vez en `API Keys`, haz click sobre el botón `Crear`, define un nombre para tu token y configura los permisos necesarios.
    Cuando presiones el botón `Guardar`, se mostrará una clave que solo se mostrará una sola vez.
  </Step>
</Steps>

Para realizar la integración más básica solo es necesario que uses el endpoint de Menu
