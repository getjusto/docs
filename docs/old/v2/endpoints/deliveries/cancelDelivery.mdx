# Ejemplo

**Endpoint:**  
`POST /cancelDelivery`

```json
{
  "deliveryId": "gAMR9R63yaRd5xmBK",
  "externalId": "P13G3-extra"
}
```
 ### Respuesta
```json
{
  "status": "ok",
  "data": {
    <canceledDeliveries>
  }
}
```
