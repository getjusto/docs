### Descripción

Este método permite gestionar el cierre temporal de una tienda o local. Resulta esencial para gestionar el período activo del local y prevenir que los clientes realicen pedidos cuando el local no se encuentra operativo. Asimismo, posibilita la cancelación de un cierre temporal que se encontraba programado.
