title: "Seleccionar el tipo de despacho"
description: "Usa este metodo para seleccionar el tipo de despacho"
openapi: Post /orders/{id}/setOrderManagedDelivery
---

### Descripción

Este método proporciona la capacidad de determinar dinámicamente quién será responsable de la entrega de un pedido, ya sea Justo o el establecimiento local.
