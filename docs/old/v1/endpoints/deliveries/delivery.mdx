### Descripción

Este método posibilita la consulta de la información y el estado actual de un despacho (`Delivery`) específico. Esta función es esencial para el seguimiento y la gestión de entregas, ya que permite que tanto el usuario como el sistema obtengan los datos actualizados sobre cada proceso de despacho.
