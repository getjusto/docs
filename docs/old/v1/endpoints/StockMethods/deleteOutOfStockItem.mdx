openapi: "Post /deleteOutOfStockItem"
---

### Descripción

Este método permite reponer un artículo que previamente fue marcado como producto agotado mediante el método addOutOfStockItem. Al proporcionar el identificador único del artículo, el sistema lo eliminará de la lista de productos agotados.
