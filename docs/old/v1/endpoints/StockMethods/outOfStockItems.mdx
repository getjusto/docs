## Ejemplo de llamada con un local

**Endpoint:**  
`POST /outOfStockItems`

```json
{
    "storeId": "xKoiiqdtKGaRA2ve4"
}
```

## Ejemplo de llamada con lista de locales

**Endpoint:**  
`POST /outOfStockItems`

```json
{

        "xKoiiqdtKGaRA2ve4",
        "dq3Ts2vq8a2bPMThm",
        "fhZxHDCXRwgyoXRDn"
    ]
}
```
