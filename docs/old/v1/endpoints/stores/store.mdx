---
title: "Obtener un local"
description: "Usa este método para obtener la información de un local"
openapi: "Post /store"
---

Entrega la información de un local

### Ejemplo

#### Llamada

```json 
{
  "storeId": "Li4eEa3W2hiuGGCFa"
}
```

#### Respuesta

```json 
{
  "status": "ok",
  "data": {
    "_id": "Li4eEa3W2hiuGGCFa",
    "websiteId": "ErCkmcyCEovseJq5E",
    "name": "Colón",
    ...
  }
}
```
