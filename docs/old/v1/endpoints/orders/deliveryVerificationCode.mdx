## Descripción

Permite definir un código de verificación que deberá ser entregado por el cliente al repartidor al momento de recibir la orden.

### Ejemplo

Llamada

```json 
{
  "orderId": "vsYr9ou4ew3RhA4X3",
  "deliveryVerificationCode": "PERRO"
}
```

Respuesta

```json 
{
  "status": "ok",
  "data": {
    "message": "Delivery verification code set successfully",
    "orderId": "vsYr9ou4ew3RhA4X3",
    "deliveryVerificationCode": "PERRO"
  }
}
```
