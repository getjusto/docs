## Descripción

Entrega la información de un pedido por el id o el código

### Ejemplo

Llamada

**Endpoint:**  
`POST /order` 

`Content-Type: application/json`

```json 
{
  "orderId": "xabAso2p3pKkLMTuz"
}
```

Respuesta

```json 
{
  "status": "ok",
  "data": {
    <Order>
  }
}
```
